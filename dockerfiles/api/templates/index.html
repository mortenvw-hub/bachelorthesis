LOD@VZG

This site provides an API to the LOD@VZG Knowledge Graph. The Knowledge Graph integrates metadata about libraries provided in the Common Library Network (GBV). The following APIs are provided:

  • SPARQL query API to the RDF triple store at /api/sparql (see SPARQLExplorer for inspection).

  • Linked Open Data (LOD) about collections of library metadata in the graph for collection URIs in the http://localhost:8000/collection/.

The project is based on the NFDI4Objects Knowledge Graph. Please consult the GitHub repository of the LOD@VZG Knowledge Graph for more information about the project.
